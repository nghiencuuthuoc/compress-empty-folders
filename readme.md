
### Cách dùng:

1. **Mặc định (nén + xoá)**

   ```bash
   python compress_empty_folders.py
   ```

   hoặc

   ```bash
   python compress_empty_folders.py -i "E:\DrugDev"
   ```

2. **Chỉ xoá**

   ```bash
   python compress_empty_folders.py -i "E:\DrugDev" --delete-only
   ```

3. **Chỉ nén**

   ```bash
   python compress_empty_folders.py -i "E:\DrugDev" --zip-only
   ```
